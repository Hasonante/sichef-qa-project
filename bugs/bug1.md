# Bug 1

## Título
Los datos no se sincronizan entre iPad y iPhone

## Pasos para reproducir
1. Abrir la app en iPad  
2. Iniciar sesión  
3. Añadir datos  
4. Abrir en iPhone  
5. Iniciar sesión  

## Resultado esperado
Los datos deberían aparecer en ambos dispositivos

## Resultado actual
Los datos no aparecen o se duplican

## Severidad
Alta

## Dispositivos
iPad / iPhone
